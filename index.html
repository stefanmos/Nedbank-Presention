Sketch Libriary V2

A libriary to help you build user-centric products with more consistency so we can focus on improve banking rather than doing repetive tasks.

  • Less modifiers

    Components will have less modifiers and more states inside the component list.

  • Responsive componets

    Most components will have their states already built and a select few will still work with modifiers

  • Typography

    Typography is now part of the Sketch library and can be pulled from the library styles

  • Finding what you need

    Componets are a lot easier to find with a clear structure.

  • Same as design system

    Components now correlate with what is seen on the design system.

  • Updates

    The library will not be worked on whilst it is live. It will be updated weekly on a specific day of the week to prevent components from breaking.

  • Emergencies

    In the rare case of an emergency update we will notify you and remind you to back up your document before updating.

    More improvements and components to be added...
